لتشغيل السيرفر الخاص بالبرنامج يجب عليك إتباع بعض الخطوات:

1- قم بتسطيب node-v18.18.0-x64.msi
2- قم بتسطيب mongodb-windows-x86_64-6.0.6-signed.msi
3- فتح cmd ثم الذهاب إلى مسار المشروع
4- كتابة الأمر npm install (يلزم الإتصال بالإنترنت في حال عدم وجود فولدر node modules)
5- كتابة الأمر node service.js

عند رفع قاعدة البيانات يجب رفع ملف excel بصيغة xlsx
يحتوي على الأعمدة: الكود - الاسم - الوصف - الرصيد - سعر البيع
